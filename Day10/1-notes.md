## Function with outputs  

#### Syntax:  
def my_func():  
    result= x*y  
    return result  

Calling it:  
    output=my_func()  

## Docstrings  
- It shows a description of what the function is used for while hovering over it  
- docstrings has to go on the first line inside the function  
- It is written inbetween 3 quotation marks
